## Deployment

### Podman Compose

```yaml
services:
  cloudflared:
    image: ghcr.io/daemonless/cloudflared:latest
    container_name: cloudflared
    environment:
      - TUNNEL_TOKEN=YOUR_CLOUDFLARE_TOKEN_HERE
      - TUNNEL_METRICS=0.0.0.0:2000
    ports:
      - 2000:2000
    restart: unless-stopped
```
